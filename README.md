# Sinergia Tienda Saludable


##  Descripción
Sinergia es una aplicación web de ecommerce enfocada en la venta de productos saludables como snacks, bebidas, suplementos y más.

## Funcionalidades
- Página de inicio (Home)
- Sistema de Login
- Página de Registro
- Navegación mediante Navbar
- Categorías de productos
- Footer con información de contacto

##  Tecnologías utilizadas
- HTML5
- CSS3
- Deploy en Vercel

## 📂 Estructura del proyecto
- index.html
- login.html
- register.html
- snacks.html
- Sin_Tacc.html
- suplementos.html
- /css/style.css
- logo.png
